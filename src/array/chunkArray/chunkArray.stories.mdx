# chunkArray


The set number is generated by the number applied to the array.  
세팅한 숫자로 배열안에 적용한 갯수의 배열로 생성된다. 

### Example

```js
import { chunkArray } from 'doum';

console.log(chunkArray([1, 2, 3], 2)); // [[1,2],[3]]
console.log(chunkArray([1, 2, 3], 1)); // [[1],[2],[3]]
```

### Reference

```js
function chunkArray<T>(arr: T[], num: number): any[T]
```

### Importing

```js
import { chunkArray } from 'doum';
```

### Arguments

- **arr** _`T[]`_  나눌 원본 배열
- **num** _`number`_  나눌 갯수


### Return
- **새로 나누어진 배열** _`any[T]`_
