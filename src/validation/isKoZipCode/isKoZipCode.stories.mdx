# isKoZipCode

Korean Road Name Address Zip Code(Postal Code)  
Verify that the first 2 digits are national basic zone numbers and check only if the remaining 3 digits are numbers.  
한국 도로명주소 우편번호 - 앞에 2자리가 [국가기초구역번호](https://ko.wikipedia.org/wiki/%EA%B5%AD%EA%B0%80%EA%B8%B0%EC%B4%88%EA%B5%AC%EC%97%AD%EB%B2%88%ED%98%B8)인지 검증하고 나머지 3자리는 숫자인지만 체크합니다.

### Example

```js
import { isKoZipCode } from 'doum';

const zipCode = '';
const zipCode2 = '30062';
const zipCode3 = '00001';
const zipCode4 = '64123';
const zipCode5 = '63086';

console.log('zipCode', isKoZipCode(zipCode)); // zipCode false
console.log('zipCode2', isKoZipCode(zipCode2)); // zipCode2 true
console.log('zipCode3', isKoZipCode(zipCode3)); // zipCode3 false
console.log('zipCode4', isKoZipCode(zipCode4)); // zipCode4 false
console.log('zipCode5', isKoZipCode(zipCode5)); // zipCode5 true
```
